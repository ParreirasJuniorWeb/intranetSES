# Versão 2.0 do Intranet SES (HTML, CSS, Bootstrap 5, React, NodeJS)
## Aplicação feita com o intuito de facilitar a comunicação do setor de TI com os funcionarios disponibilizando materiais de apoio.

```
rodando o projeto:
npm install
```
